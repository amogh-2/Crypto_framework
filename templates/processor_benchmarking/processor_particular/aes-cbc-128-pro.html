AES-CBC (128-bit Key) Encryption

AES (Advanced Encryption Standard) in Cipher Block Chaining (CBC) mode is a symmetric encryption algorithm that encrypts data in fixed-size blocks. Each block is XORed with the previous encrypted block to enhance security. A 128-bit key is commonly used for balance between security and performance.
